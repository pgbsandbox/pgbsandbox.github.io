File server

The File Server top menu may or may not be configured for your deployment. It provides access to a simple remote file server. You may be using a different integration for storing your topics.

Definition

A file server is any central storage mechanism to hold files or data from a group of people, such as a department, division, or company.

How AuthorBridge uses file servers

Customers can leverage an existing corporate file server that may already be used to store other documents or topics. You can connect AuthorBridge directly to that file server.

The advantage of using a filer server is that all content is saved to a common location, no matter who creates it. People can access topics made by others as well as their own; no topic ever gets lost on an individual's hard drive.

You can open, save, and save as from File Server.

Contact Stilo if you have a file server you want to connect to.
