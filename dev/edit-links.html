Change a link

You can change a link after you have created it.

If you need to change the properties of a link, such as the location that you link to, you can modify an existing link rather than deleting the old one and creating a new one.

  1. In your topic, click once anywhere on the link.
    The hover text saying Link displays.
  2. Select the Link that hovers.
  3. Select the edit link button.
    The link properties display.
    URL
    The link URL is the relative path to the topic file name and, optionally, a part inside the topic.

    Link syntax: /relative-path/file_name.dita#topic_id/part_id

    Internal
    Link to another topic in the documentation set.
    External
    Link to something outside of the documentation set, like a website.
    Scope
    Local: A topic that is in the same family as the topic you place the link in.
    External: Link to a topic or site that is outside the family of the topic you are placing the link in.
    Peer: Link to a topic that will eventually be in the same family as the topic you are placing the link in.
    Format
    The format of the place you are linking to. If it is an internal link, the value is dita. Other valid values are html and pdf.
    Type
    Used only for internal links. Indicate the topic type you are linking to: concept, task, reference, or topic.
  4. Make your modifications.
  5. Select OK.
