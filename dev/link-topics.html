Link from one topic to another

Links are very distracting to readers. The key is to insert a link only when absolutely required to keep them to a minimum. Then readers will know that it is not just tangentially relevant information but really important information.

Both topics should be located on a file server or CMS before you can create a link between them. It is possible to link between topics that are not on a file server or CMS but it is not recommended.

A link from one topic to another lets you connect information from one topic to another. This, in turns, lets a reader find associated information without overloading any one particular topic with too much information. While you can link to any element of the target topic, it is generally best to link to the whole topic, or to something inside that has a title of caption, such as a table or figure. When doing so, link to the main structure, not to its title or caption.

Copy the link target to you clipboard.

  1. Open the topic you want to create a link to.
    This is the target topic.
  2. Right-click on the content you want to link to.
  3. Select Choose Link Target and then select the content you want to link to.
    As you hover over the presented options, the background in the editor is adjusted to indicate the corresponding region, helping you to decide what you want to link to.
  4. Note whether or not the the dialog boxes informs you that you will need to save this topic.
    It is possible that an id attribute was inserted into the link target location, and this is why you may need to save this topic
  5. Press the Copy.
  6. Press the Close button.
  7. Save this topic if you were requested to do so.

Paste the link target to create a link.

  1. Open the topic in which you want to place the link.
    This is the source topic. It contains the clickable link.
  2. Create the link
    • If you are linking to something with a title, place the cursor at the location you would like the link to be inserted and, without making a text selection, press CTRL + V.
    • Otherwise it is best to type some appropriate link text, select this text, and then press CTRL + V.
A link has been added to you topic. The link text is underlined.
