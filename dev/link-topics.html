Link from one topic to another

Links are very distracting to readers. The key is to insert a link only when absolutely required to keep them to a minimum. Then readers will know that it is not just tangentially relevant information but really important information.

Both topics should be located on a file server or CMS before you can create a link between them. It is possible to link between topics that are not on a file server or CMS but it is not recommended.

A link from one topic to another lets you connect information from one topic to another. This, in turns, lets a reader find associated information without overloading any one particular topic with too much information.

  1. Open the topic you want to create a link to.
    This is the target topic.
  2. Right-click on the content you want to link to.
  3. Select Choose Link Target and then select the content you want to link to.
    Usually, you link to a section, step, or paragraph. If you want to link to the entire topic, select the topic type (task, concept, or reference).
  4. Save this topic.
  5. Following the prompts, press CTRL + C to copy the link information.
  6. Open the topic in which you want to place the link.
    This is the source topic. It contains the clickable link.
  7. Put cursor where you want to create link.
  8. Type some appropriate text for the link and select this text.
  9. Press CTRL + V to paste your link.
When the content is published, the link text automatically changes to the title of the topic or section that you are linking to.
