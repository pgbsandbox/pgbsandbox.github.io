Link to a website

You can link to any website on the internet from your topics.

  1. In your browser, open the website page you want to link to.
  2. Copy the URL from the address bar.
  3. In your topic, highlight the text you want to add a link to.
  4. From the toolbar, select Insert Link.
  5. Select External.
  6. Paste your URL into the field.
  7. In the format field, type html.
