Control line breaks

Fixed font functionality preserves line breaks for content like code samples, where the line breaks must be very specific.

Line breaks can be any of five types:
Preformatted
A generic tag for any content that needs preserved line breaks. Do not use if another tag describes the content better. Also changes the font to mono space.
Code block
Used for lines of program code, such as C++.
Screen console
When you want to show exactly what a screen displays without creating a graphic of it. Instead, you can use text and force the lines to break where they do on the scene you are re-creating.
Message block
For displaying any multi-lined messages in text.
Preserve line breaks
A generic tag for any content that needs preserved line breaks. Do not use if another tag describes the content better. Does not change the font.

Always choose the type that most closely applies to the content in the fixed font box.

  1. Place your cursor on an empty line, at the beginning of where you will place your fixed font content.
  2. From the toolbar, select Fixed Font.
    A fixed font box displays.
  3. Select the FF at the top of the fixed font box.
  4. Choose the fixed font option that applies to your type of content.
  5. Add your fixed font content to the box.
    Your line breaks are preserved.
  6. Exit out of the fixed font box by entering Enter on the last blank line.
