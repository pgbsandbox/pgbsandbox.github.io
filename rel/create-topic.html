Create a new topic

Create a new topic for each distinct but complete subject matter you need to write about. For example, you would create a new topic to write a task about performing a procedure that has a set of steps.

There are three main topic types to choose from: concept, task, and reference. See Topic types for more information.

  1. In the top menu, select New.
  2. Select the topic type you need.
    A new topic opens for you with default, placeholder parts and text. The Guidance panel opens on the left. The toolbar displays above.
  3. Add your content, using the Guidance panel and toolbar options.
  4. Save or check in your topic.
    Your options for saving your work depend on what integrations you are working with.
