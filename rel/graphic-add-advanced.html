Add a graphic placeholder

You can add a placeholder for a graphic that you do not yet have access to.

Sometimes, you know that a graphic needs to be inserted even if you do not currently have the ability to browse for it, either because it does not yet exist or because you are writing a non-rooted topic. You can insert a placeholder that will resolve to a graphic with that file name at a later date.

  1. From the toolbar, select Insert Image.
  2. Select No image and type or paste the relative path and image name.
  3. Select Adjust and set the properties of the image.
    • Placement:
      • In-line: For images that are part of a sentence.
      • Left: To left align an image. For images that are on their own line.
      • Center: To center the image on the page. For images that are on their own line.
      • Right: To right align an image. For images that are on their own line.
    • Scale:
      • Scale: The percent size of the original image. 50% makes the image half the size of the source image size.
      • Width and height: Dimensions of the image.
    • Alt text: Text that displays when the topic is published to HTML when someone hovers over the image and that a screen reader reads.
  4. Click OK.
