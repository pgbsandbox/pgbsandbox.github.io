Choose a table type

Different table types are available for specific kinds of content.

  1. In your table, right click and select Choose table element.
    The available types display above the table content.
  2. Select your table type.
    • Definition list: A two-column table that displays terms in the left column and their definitions in the right column. No header.
    • General CALS table: A flexible table used for most content.
    • Simple table: A table that organizes and classifies items, like a parts list or contacts list.
    • Choice table: Only available in tasks after a step, choice tables provide decision tree options and their descriptions, usually in an If ... (first column) Then ... (second column) format.
    • Parameter list: A list of software parameters, commands, or options that each have a description and possibly some usage examples.
