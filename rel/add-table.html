Add a table

Tables are excellent way to organize and display complex information.

  1. In your topic, place your cursor on a new, empty line.
  2. From the toolbar, select Insert Table.
  3. Specify the number of rows and columns.
    You can adjust these later.
  4. Specify
    • whether the table will beÂ  the full width of the page
    • where on the page it will be aligned
    • where a frame will be displayed
    • whether column and row separators will be displayed

  5. (Optional) Right click anywhere in the table and select Choose table element.
    This will allow you to specify which type of table you want to create. Choose from
    • general CALs table
    • choice table
    • simple table

    Which of these is available will depend upon the topic type you are working in. The default will be a simple table.

  6. (Optional) Right click anywhere in the table and select Table Options.
    From here you can modify the display settings for your table.
  7. (Optional) Right click anywhere on the table and select Row or Column to set a heading row or column

The table is inserted, with all header cells in blue.
