Add a figure

A figure is a container part that usually holds a graphic and a title.

If you need to include some specific information about a graphic, such as a caption or a description, you can add a figure. The graphic is optional, so you can also add a figure simply to group a small amount of content together under a title. For example, you may want to add a title above an example of code block.

  1. From the toolbar, select Figure.
    A figure part is inserted, with sub-parts. The Guidance panel displays all the parts.
  2. Use the Guidance panel to add the parts of the figure that you want.
  3. (Optional) Select Insert Image or add another part that you want.
    An image is normally added to a figure but is optional. You could also add a code block or some other chunk of information that needs to be grouped together under a title.
