Remove a link

If your link no longer applies, you can remove it.

  1. In your topic, click once anywhere on the link.
    The hover text saying Link displays.
  2. Select the Link that hovers.
  3. Select the X.
    Tip: Use the undo button to bring the link back if you deleted it in error.
    The text is no longer a link. The text remains and is ready to be removed.
