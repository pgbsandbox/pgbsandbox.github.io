Reuse content from another topic

Content reuse is a very helpful tool for writers. There is no need to copy common text between topics. It can be stored in one place and reused in many. If you ever need to update the text, you will only have to do it in one place.

Both topics should be located on a file server or CMS before you can create a link between them.

Copy the the content to reuse

  1. Open the topic that contains the content you want to reuse.
  2. Right-click on the content you want to link to.
  3. Select Choose Content to Reuse and then select the content you want.
    A dialog box will appear with further instructions.
  4. Following the prompts, press CTRL + C to copy the content for reuse to your clipboard.
  5. Save this topic.

Paste the content to reuse

  1. Open the topic in which you want to reuse the content.
  2. Put caret where you like and press CTRL + V to paste your link.
When the content is published, the reused text will automatically be pulled in. In AuthorBridge, the reused text will be read-only.
